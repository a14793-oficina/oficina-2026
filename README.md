# 🐍 Exercícios de Python — Estruturas Condicionais

Este repositório contém uma coleção de exercícios introdutórios em **Python**, focados no uso de **estruturas condicionais (`if`, `elif`, `else`)**. Estes exercícios fazem parte do meu percurso no curso **Técnico de Gestão e Programação de Sistemas Informáticos (TGPSI)** e têm como objetivo treinar lógica de programação e tomada de decisões em código.

---

## 📘 Conteúdo dos Exercícios

### 🔹 B1 — Número Positivo ou Negativo
O programa pede um número ao utilizador e indica se é **positivo** ou **negativo**.

### 🔹 B2 — Maior de Idade
Recebe a idade do utilizador e informa se é **maior** ou **menor de idade**.

### 🔹 B3 — Número Par ou Ímpar
Lê um número inteiro e determina se é **par** ou **ímpar**.

### 🔹 B4 — Comparação de Dois Números
Compara dois valores introduzidos e mostra qual é o **maior**.

### 🔹 B5 — Password Simples
Verifica se a password introduzida corresponde à password correta.

---

## 📘 Exercícios Intermédios (I)

### 🔹 I1 — Classificação de Nota
Classifica uma nota entre 0 e 20 como:
- Reprovado  
- Suficiente  
- Bom  
- Excelente  

### 🔹 I2 — Classificação de Idade
Classifica a idade em:
- Criança  
- Jovem  
- Adulto  
- Sénior  

### 🔹 I3 — Múltiplo de 3 e 5
Verifica se um número é múltiplo de:
- 3  
- 5  
- Ambos  
- Nenhum  

### 🔹 I4 — Login com Utilizador e Password
Valida um login simples com utilizador e password.

### 🔹 I5 — Número dentro de intervalo
Verifica se um número está entre **10 e 20**.

---

## 📘 Exercícios Avançados (A)

### 🔹 A1 — Sistema Multibanco Simples
Simula um levantamento, verificando se o valor pedido é inferior ao saldo.

### 🔹 A2 — Maior de Quatro Números
Compara quatro números e determina o **maior**.

### 🔹 A3 — Cálculo de IMC (Simplificado)
Calcula o **Índice de Massa Corporal** e classifica o resultado.

### 🔹 A4 — Sistema de Desconto
Aplica descontos com base no valor da compra:
- ≥ 100€ → 10%  
- ≥ 50€ → 5%  
- < 50€ → sem desconto  

### 🔹 A5 — Verificação de Ano Bissexto
Determina se um ano é **bissexto** ou **normal**.
